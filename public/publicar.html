Formulario de subida de archivos

Reglas

Recordad que el título debe coincidir con el nombre del archivo a subir, excepto la extensión

El archivo debe ser de texto plano, se admiten los archivos con la extensión txt, md u html, este útimo sin estilos css ni código javascript

La publicación se hará en la Bibliotéca, una vez comprobado que el contenido del archivo no tiene contenido malicioso, y estará disponible en la biblioteca de la página para ser debatido por comentarios, o por el chat, para cualquier persona que lo desee.
